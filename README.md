# Smart Study Tracker using Prolog

This project is a simple logic-based Study Tracker built using Prolog. It demonstrates how students can manage subjects, assignments, and study progress using rule-based reasoning.

Unlike modern applications that rely on databases or machine learning, this project uses logical facts and rules to simulate intelligent task tracking and progress evaluation.

## What This Project Does

* Stores subjects in a knowledge base
* Stores study tasks with deadlines
* Tracks pending and completed assignments
* Calculates overall study progress percentage
* Displays task information automatically

## How It Works (Simple Explanation)

Subjects are stored as facts in Prolog
Tasks are linked to subjects with deadlines
Each task has a status (pending or completed)
Rules calculate completed and pending task counts
Progress percentage is calculated automatically
